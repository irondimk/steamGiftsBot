# steamGiftsBot
A program for automatically joining distribution groups on the site steamgifts.com.
For work, you need to translate the authentication via mail to mail.ru
Rename the following lines in the Form1.cs file

string log = ""; //steam login
string pas = ""; //steam pass

string mlog = ""; //mail.ru login
string mpas = ""; //mail.ru pass

insert your data between the quotes
